2002-Cognitve Models of the Brain-Prof. Wang

Posted on 2019-03-29

Paper Title: “Cognitive Models of the Brain, Prof. Yingxu Wang, 2002”
Questions proposed by myself:

  1. Is the memory and computation of human seperate?
  2. Is cognitive machine in need of a rest like human’s sleep to develop, improve, or reconfigure its memory?
  3. What is the difference between cognitive model of the brain and the functional model of the brain?
  4. Does we human brain store information and knowledge in the similar representation structure?
  5. The representation of LTM or OAR model?

Highlights in this paper:

  1. Memory is the foundation of any intelligence and maintaining a stable state, Why ?.
  2. Genome research, a bottom level research, can only explain things at Natural Intelligence operating systemt (NI-OS) level, like inherited life function, while cognitive informatics, a high level research, can explian things at NI-App level, like acquired lift functions.
  3. The number of neurons in human brian is variable over time, see Fig.2. Human brain contains about 100 billion neurons and each neuron possesses thousands of synapes connecting to other neurons. Besides, the number of neurons in human brain initially grows(0-8 months), then reduces (8-144 months) to a stalbe state (144months - death).
  4. Things matter is the magnitude of neural networks and their cocurrent behaviors.
  5. Structure of Memory and corresponding function model: Sensory buffer memory (Queues), Short term memory (stacks) (!!!reasonable ???), long-term memory (dynamic, hiearchical and partially interconnected neural networks), Action buffer memory (Queues).
  6. Cognitive model of Long-term Memory(LTM): object-attribute-relation (OAR) model—Fig.4, object and attribute are all neurons while relations are synapes.
  7. Some constraints or properties of LTM:
    • LTM is dynamic
    • It is directed
    • It contains loops
    • It can generate new relations(connections)
    • It can be reconfigured
    • It can be traversed or searched
    • It cannot be sort
  8. Brain and natrual intelligence(NI) is a real-time information processing system.
  9. Computational model of the brain: NI-sys = NI-OS || NI-App, || is a parallel relation, see Table 4:
    • NI-OS (thalamus, willingness-driven processes) represents the inherited life functions, like sensation (Vision, Audition, etc.) and sub-conscious life functions (Memory, Feeling, Desires and Personality).
    • NI-App(cerebral cortex, event- and time- driven processes)is the acquired or developed functions.
    • Information can be classfied into event-driven, time-driven, desire-driven.
    • It is a passive system.
  10. Functional model of the brain, see Fig.6.
  11. Cognitive model of the brain, see Fig.7.
  12. The LTM is built by searching and analyzing the contents of STM and selecting useful infromation into LTM.
    • An OAR was most frequently used
    • A new OAR is relating to an existing one
    • An OAR is given special attention
  13. LTM establishment: identify, produce links, create a new OAR cluster!!! if no existing or related konwledge culster.
  14. LTM: cognition (identify useful information), retention (store knowledge), reproduction(retrieve knowledge)
